
This is a console-based Stock Market Simulator coded in Java where users start with $10,000 
and can do four actions: buy stock, sell stocks, wait for the next day seeing how the stock value 
changes randomly, or exit the game. The language is Java and it uses arrays,loops, random number generation, user input, and conditionals.

How to Run

1)Make sure Java is installed on your computer. Check by opening terminal/command prompt and typing java -version. If nothing shows, download Java from java.com.
2)Go to the repository and click on GPACalculator.jar, then click Download raw file to download it. 
3)Open your terminal (Mac/Linux) or Command Prompt (Windows). 
4)Navigate to the folder where you downloaded the file. For example: cd Downloads 
5)Run this command: java -jar GPACalculator.jar The program will start in your terminal
